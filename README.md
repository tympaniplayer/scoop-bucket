# Neurohelmet Scoop Bucket

A [Scoop](https://scoop.sh) bucket for [Neurohelmet](https://github.com/tympaniplayer/neurohelmet) —
a keyboard-driven terminal BattleTech record-sheet tracker.

## Install

```powershell
scoop bucket add neurohelmet https://github.com/tympaniplayer/scoop-bucket
scoop install neurohelmet
```

Update with `scoop update neurohelmet`.

## Notes

`bucket/neurohelmet.json` is generated automatically on each Neurohelmet release.
Don't edit it by hand — it's overwritten on the next release.
